# 📝 Lista de Tarefas - JavaScript Puro

Aplicação de lista de tarefas desenvolvida com JavaScript moderno, sem frameworks.


## 🌐 Acesse o projeto online

[🔗 Clique aqui para ver funcionando](https://francisco-caio.github.io/todo-list-js/)


## 🚀 Tecnologias

- **JavaScript ES6+** (Classes, Arrow Functions, Spread Operator, Async/Await)
- **localStorage** (persistência de dados no navegador)
- **Métodos de Array** (map, filter, sort)
- **HTML5 & CSS3**

## ✨ Funcionalidades

- ✅ Adicionar tarefas
- ✅ Marcar como concluída/não concluída
- ✅ Remover tarefas
- ✅ Persistência com localStorage (dados ficam salvos mesmo fechando o navegador)
- ✅ Contador de estatísticas (total, concluídas, pendentes)

## 🎯 Conceitos Aplicados

- Arquitetura com **Classes ES6**
- **CRUD** completo (Create, Read, Update, Delete)
- **Imutabilidade** com Spread Operator
- **MVC** simplificado (Model: dados, View: DOM, Controller: lógica)

## 🖥️ Como usar

1. Clone o repositório
2. Abra o `index.html` no navegador
3. Adicione suas tarefas!
